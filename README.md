# KaryoScope Analysis

Documentation and analysis tools for KaryoScope clustering methods.

## Overview

This repository contains comprehensive documentation for KaryoScope, a graph-based method for analyzing long-read sequencing data that identifies structural patterns based on genomic feature composition and transition patterns.

## Documentation

The documentation is built using MkDocs Material and hosted on GitHub Pages.

**Live Documentation:** [https://barthel-lab.github.io/KaryoScope-analysis/](https://barthel-lab.github.io/KaryoScope-analysis/)

## Contents

- **Comprehensive Technical Guide**: Detailed explanation of the clustering algorithm, from input data to final enrichment calls
- **Analysis Scripts** (coming soon): Python scripts for running KaryoScope clustering analysis
- **Examples** (coming soon): Example workflows and tutorial notebooks

## Local Development

To build and preview the documentation locally:

```bash
# Install dependencies
pip install -r requirements.txt

# Serve documentation locally
mkdocs serve

# Build static site
mkdocs build
```

The documentation will be available at `http://127.0.0.1:8000/`
